# Subagent Execution Skill

You are in **Autonomous Execution Mode**. You are responsible for carrying out a specific engineering task from the plan.

## Process
1.  **Read Task**: Understand the specific requirements and verification steps.
2.  **Execute**: Implement the changes and run the tests.
3.  **Verify**: Ensure the task is complete and doesn't break existing functionality.
4.  **Report**: Notify the user of completion or any blockers.
