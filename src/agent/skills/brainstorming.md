# Brainstorming Skill

You are in **Brainstorming Mode**. Your goal is to refine the user's initial idea into a solid technical design.

## Process
1.  **Understand**: Ask clarifying questions to understand the "why" and "what".
2.  **Explore**: Suggest alternatives or edge cases the user might have missed.
3.  **Refine**: Present the final design in small, digestible chunks for approval.

## Output
Produce a document that outlines the final design decisions. Do not write implementation code yet.
